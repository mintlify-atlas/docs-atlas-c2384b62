## Core Technologies

### React 18

**Version**: `^18.2.0`

**Why React?**
- **Component-based architecture** aligns perfectly with Clean Architecture
- **Hooks** enable clean separation of UI and business logic
- **Large ecosystem** with excellent libraries and tooling
- **Performance** with Virtual DOM and concurrent features
- **Active community** with extensive documentation

**Key Features Used**:
- Functional components with hooks
- React Router for navigation
- Context API for global state (user session, admin auth)
- Custom hooks in Application layer (useLogin, useUserProfile, useAdmin)

```jsx
// Example: Custom hook pattern
const { validateId, loading, error } = useLogin();
```

### Vite

**Version**: `^5.0.8`

**Why Vite?**
- **Lightning-fast HMR** (Hot Module Replacement)
- **Instant server start** regardless of application size
- **Optimized builds** with Rollup under the hood
- **Native ES modules** support
- **Better developer experience** compared to Create React App

**Configuration**:

```javascript
// vite.config.js
import { defineConfig } from 'vite';
import react from '@vitejs/plugin-react';

export default defineConfig({
  plugins: [react()],
  server: {
    port: 3000,
    open: true
  },
  build: {
    outDir: 'dist',
    sourcemap: true
  }
});
```

**Scripts**:
- `npm run dev` - Start development server (port 3000)
- `npm run build` - Build for production
- `npm run preview` - Preview production build

### Tailwind CSS

## Backend & Data Layer

### Supabase

**Version**: `@supabase/supabase-js ^2.97.0`

**Why Supabase?**
- **PostgreSQL database** with full SQL capabilities
- **Real-time subscriptions** for live data updates
- **Row Level Security** built-in for data protection
- **Auto-generated REST API** from database schema
- **Authentication** with JWT tokens
- **Storage** for files and documents
- **Open source** with self-hosting option

**Client Setup**:

```javascript

import { createClient } from '@supabase/supabase-js';

const SUPABASE_URL = 'https://zhljlbuwpezwgkpwvkeu.supabase.co';
const SUPABASE_KEY = 'sb_publishable_wmcyhNSE68bZ9fW8v1PsMg_Va7sIKe-';

export const supabase = createClient(SUPABASE_URL, SUPABASE_KEY);
```

### XLSX (SheetJS)

**Version**: `^0.18.5`

**Purpose**: Excel export functionality

```javascript
import * as XLSX from 'xlsx';

const exportToExcel = (data, filename) => {
  const worksheet = XLSX.utils.json_to_sheet(data);
  const workbook = XLSX.utils.book_new();
  XLSX.utils.book_append_sheet(workbook, worksheet, 'Report');
  XLSX.writeFile(workbook, `${filename}.xlsx`);
};
```

**Why XLSX?**
- Industry standard for Excel manipulation
- Both import and export capabilities
- Supports .xlsx, .xls, .csv formats
- Works in browser (no backend needed)
- Used for monthly reports and data imports

### Axios

**Version**: `^1.6.2`

**Purpose**: HTTP client for API requests (backup to Supabase)

```javascript
import axios from 'axios';

const apiClient = axios.create({
  baseURL: import.meta.env.VITE_API_URL || 'http://localhost:3000',
  timeout: 10000,
  headers: {
    'Content-Type': 'application/json',
  }
});

// Request interceptor for JWT
apiClient.interceptors.request.use((config) => {
  const token = localStorage.getItem('access_token');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

// Response interceptor for errors
apiClient.interceptors.response.use(
  (response) => response,
  (error) => {
    if (error.response?.status === 401) {
      // Redirect to login
      window.location.href = '/login';
    }
    return Promise.reject(error);
  }
);
```

**Why Axios?**
- Clean API for HTTP requests
- Automatic JSON transformation
- Interceptors for auth tokens
- Request/response transformation
- Better error handling than fetch

## Performance Considerations

### Bundle Size Optimization

- **Tailwind purge**: Removes unused CSS (production builds less than 10KB)
- **Code splitting**: React Router automatically splits routes
- **Tree shaking**: Vite removes unused JavaScript
- **Lazy loading**: Components loaded on demand

### Runtime Performance

- **React 18 features**: Automatic batching, transitions
- **Supabase connection pooling**: Efficient database queries
- **Memoization**: React.memo, useMemo, useCallback where needed
- **Optimistic updates**: UI updates before server confirmation
