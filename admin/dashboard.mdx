## Visión General

El panel administrativo de UCC Control de Acceso proporciona una vista completa del estado del sistema, permitiendo a los administradores monitorear el acceso contingente de usuarios en tiempo real y gestionar el control de carnets institucionales.

<Info>
El dashboard se actualiza automáticamente cada 30 segundos sin afectar la vista actual del usuario.
</Info>

## Acceso al Panel

Solo **empleados autorizados** registrados en la tabla `administradores` pueden acceder al panel administrativo. El sistema valida los permisos al iniciar sesión.

```javascript title="src/modules/admin/application/hooks/useAdminDashboard.js"
const INTERVALO_REFRESCO = 30_000; // 30 segundos

export const useAdminDashboard = () => {
  const [stats, setStats] = useState(null);
  const [ultimaActualizacion, setUltimaActualizacion] = useState(null);
  
  // Refresco silencioso cada 30s — no muestra spinner
  const refrescoSilencioso = useCallback(async () => {
    try {
      const [s, u] = await Promise.all([
        repo.getStats(),
        repo.getUsuarios(),
      ]);
      setStats(s);
      setUsuarios(u);
      setUltimaActualizacion(new Date());
    } catch {
      // silencioso — no muestra error
    }
  }, []);
  
  return { stats, usuarios, loading, error, ultimaActualizacion };
};
```

El dashboard muestra cuatro tarjetas principales con métricas clave del sistema:

## Gráficas y Visualizaciones

### 1. Estado de Carnets (Gráfica Donut)

Muestra la proporción de usuarios activos vs. bloqueados en un gráfico circular con anillo central:

```jsx
const donutData = stats ? [
  { name: 'Activos',    value: stats.activos    },
  { name: 'Bloqueados', value: stats.bloqueados },
] : [];

<PieChart>
  <Pie
    data={donutData}
    innerRadius={60}
    outerRadius={85}
    paddingAngle={3}
    dataKey="value"
  >
    <Cell fill="#1B6B3A" /> {/* Verde UCC */}
    <Cell fill="#EF4444" /> {/* Rojo */}
  </Pie>
</PieChart>
```

<Note>
La etiqueta central del donut muestra el total de usuarios en el sistema.
</Note>

### 2. Reportes TIC - Últimos 7 Días

Gráfico de barras que muestra la distribución de fallas (olvido y pérdida) durante la última semana:

```jsx
<BarChart data={fallas7d}>
  <Bar dataKey="olvido"  name="Olvido"  fill="#1B6B3A" stackId="a" />
  <Bar dataKey="perdida" name="Pérdida" fill="#EF4444" stackId="a" />
</BarChart>
```

### 3. Estudiantes por Programa

Gráfico de barras horizontales que muestra la distribución de estudiantes por programa académico:

```jsx title="Distribución por programa"
<BarChart layout="vertical" data={porPrograma}>

  <XAxis type="number" allowDecimals={false} />

</BarChart>
```

## Navegación del Panel

El dashboard incluye una barra lateral (sidebar) con las siguientes secciones:

<Steps>
  <Step title="Resumen">
    Vista principal con todas las estadísticas y gráficas en tiempo real
  </Step>
  <Step title="Usuarios">
    Listado completo de usuarios con búsqueda, filtros y gestión de acceso
  </Step>
  <Step title="Semestre">

  </Step>
  <Step title="Informes">

## Estado del Sistema

El header del dashboard muestra información importante:

- **Última actualización**: Hora del último refresco de datos (formato HH:mm:ss)
- **Estado del sistema**: "Activo" con el número de estudiantes habilitados
- **Errores**: Alertas visuales si hay problemas de conexión

```jsx
{ultimaActualizacion && (
  <span className="text-xs text-gray-400">
    Actualizado: {ultimaActualizacion.toLocaleTimeString('es-CO', {
      hour: '2-digit',
      minute: '2-digit',
      second: '2-digit'
    })}
  </span>
)}

{error && (
  <span className="text-xs text-red-500 bg-red-50 px-3 py-1.5 rounded-lg">
    ⚠️ {error}
  </span>
)}
```
