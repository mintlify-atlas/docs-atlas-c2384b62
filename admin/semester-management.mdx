## Descripción General

El módulo de gestión de semestre permite a los administradores iniciar un nuevo período académico mediante la carga masiva de datos a través de archivos CSV, reemplazando los datos del semestre anterior de forma controlada y guiada.

<Warning>
Al iniciar un nuevo semestre, **todos los datos del semestre anterior se eliminarán** de forma permanente. Asegúrate de exportar los reportes finales antes de proceder.
</Warning>

## Proceso de Inicio de Semestre

El sistema utiliza un wizard de 6 pasos para guiar al administrador de forma segura:

<Steps>
  <Step title="Paso 1: Información del Semestre">
    Configura el nombre, fecha de inicio y fecha de fin del nuevo semestre
  </Step>
  <Step title="Paso 2: CSV de Usuarios (Obligatorio)">
    Carga el archivo con los datos base de todos los usuarios
  </Step>
  <Step title="Paso 3: CSV de Estudiantes (Opcional)">
    Asigna el rol de estudiante y programas académicos
  </Step>
  <Step title="Paso 4: CSV de Empleados (Opcional)">
    Asigna el rol de empleado con cargos y dependencias
  </Step>
  <Step title="Paso 5: CSV de Contratistas (Opcional)">
    Asigna el rol de contratista con empresas proveedoras
  </Step>
  <Step title="Paso 6: Confirmación y Procesamiento">
    Revisa el resumen y confirma el inicio del nuevo semestre
  </Step>
</Steps>

```jsx title="src/modules/admin/presentation/components/views/CargarCSVView.jsx"
  <div className="bg-white rounded-xl border shadow-sm px-6 py-4">
    <div className="flex items-center">

        const done = i < pasoActual;

        return (
          <React.Fragment key={i}>
            <div className="flex flex-col items-center">
              <div className={`w-8 h-8 rounded-full flex items-center justify-center ${

                'bg-gray-100 text-gray-400'
              }`}>
                {done ? '✓' : i + 1}
              </div>

                active ? 'text-[#1B4332]' : done ? 'text-green-700' : 'text-gray-400'

            </div>
            {i < NOMBRES_PASO.length - 1 && (

              }`} />

          </React.Fragment>

      })}
    </div>
  </div>
```


### Validación

El sistema valida que:
- El nombre del semestre no esté vacío
- Ambas fechas estén completas
- La fecha de fin sea posterior a la fecha de inicio

```javascript
const semestreValido =

  semestre.fechaInicio &&
  semestre.fechaFin &&
  semestre.fechaFin > semestre.fechaInicio;
```

### Parser de CSV

El sistema incluye un parser que maneja diferentes formatos:

```javascript
function parsearCSV(texto) {
  // Eliminar BOM (Excel UTF-8)
  const textoLimpio = texto.replace(/^\uFEFF/, '').trim();
  const lineas = textoLimpio.split('\n');
  
  // Soportar separador ; (Excel español) o , (estándar)
  const separador = lineas[0].includes(';') ? ';' : ',';
  
  const encabezados = lineas[0]
    .split(separador)
    .map(h => h.trim().replace(/\r/g, '').replace(/"/g, ''));
  
  return lineas.slice(1)
    .map(linea => {
      const valores = linea
        .split(separador)
        .map(v => v.trim().replace(/\r/g, '').replace(/"/g, ''));
      
      const fila = {};
      encabezados.forEach((h, i) => {
        fila[h] = valores[i] ?? '';
      });
      return fila;
    })
    .filter(f => f.id_institucional); // Solo filas con ID válido
}
```

<Note>
El parser automáticamente detecta si el archivo usa comas (`,`) o punto y coma (`;`) como separador, y elimina el BOM que agrega Excel en archivos UTF-8.
</Note>

## Procesamiento del Semestre

Al confirmar, el sistema ejecuta el proceso completo:

```javascript
const handleEmpezarSemestre = async () => {
  setCargando(true);
  setError(null);
  
  try {
    // 1. Crear el nuevo semestre (limpia datos anteriores)
    await repo.iniciarNuevoSemestre(
      semestre.nombre.trim(),
      semestre.fechaInicio,
      semestre.fechaFin
    );
    
    // 2. Cargar cada CSV en orden
    const res = {};
    if (staged.usuarios?.length) {
      res.usuarios = await repo.cargarUsuarios(staged.usuarios);
    }
    if (staged.estudiantes?.length) {
      res.estudiantes = await repo.cargarEstudiantes(staged.estudiantes);
    }
    if (staged.empleados?.length) {
      res.empleados = await repo.cargarEmpleados(staged.empleados);
    }
    if (staged.contratistas?.length) {
      res.contratistas = await repo.cargarContratistas(staged.contratistas);
    }
    
    setResultado(res);
    setPaso(6); // Ir a pantalla de éxito
  } catch (err) {
    setError(err.message ?? 'Error al iniciar el semestre');
  } finally {
    setCargando(false);
  }
};
```

## Recomendaciones

<Note>
**Antes de iniciar un nuevo semestre:**
1. Exporta todos los reportes del semestre actual (diario, semanal, mensual, semestral)
2. Guarda copias de seguridad de los archivos Excel generados
3. Verifica que todos los CSV tengan el formato correcto
4. Prueba con un archivo pequeño primero si no estás seguro
</Note>
