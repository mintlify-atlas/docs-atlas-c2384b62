# Welcome to UCC Control de Acceso

UCC Control de Acceso is a comprehensive access control system designed for the Universidad Cooperativa de Colombia. The system manages contingent entry for users who **do not carry their physical ID card**, tracking access failures, implementing automated blocking rules, and providing administrative oversight.

<Note>
This application is **only** used when a user does **not have their physical card**. It does not register normal access events.
</Note>

## System Overview

The UCC Control de Acceso system provides three main user flows:

1. **User Login** — Users enter their institutional ID to access their profile and report missing cards
2. **Failure Reporting** — Users select the reason for not having their card (forgotten, lost, or stolen)
3. **Administrative Dashboard** — Authorized employees manage user access, generate reports, and handle semester transitions

<Warning>
Users are automatically blocked after 4 "forgotten card" incidents or immediately upon reporting a lost or stolen card.
</Warning>

## Built With Modern Technology

- **Frontend:** React 18 with Vite, Tailwind CSS, and Clean Architecture
- **Backend:** Supabase (PostgreSQL) with Row Level Security
- **Authentication:** Institutional ID-based validation
- **Reporting:** Excel export with XLSX library
- **Design:** UCC corporate colors (Blue #003DA5 and Orange #FF6B35)
